# How does javascript works ?

JavaScript is a high level programming language. When we write JavaScript code it goes to the browser.The computerbasically does not understand Raw JavaScript code. For this, high level language has to be converted tomachinelanguage. And there is a mechanism in the browser to do this. That mechanism is basically called JavaScript engine. Thefunction of this engine is to convert raw JavaScript code to machine language. Different browsers havedifferent typesof equipment. But all browsers follow the same script. This means that after our code goes to theengine, the browsersends the code to the JavaScript engine. The engine then compiles the codes and converts them intomachine language. Andonly then can the computer understand our instructions. And later we see it on the display.

# Local storage & Session storage defferent ?

When a user visits a website, some of his data has to be stored for later use. And for this there are basically two types of storage. One is local storage and the other is session storage. If data is stored in the local storage, the data remains even if the user closes his browser tab, website, browser and even computer. That means local storage data will remain there until it is deleted. On the other hand, session storage is temporary. This means that the data will be removed from the session storage only when the user closes the website or browser.
